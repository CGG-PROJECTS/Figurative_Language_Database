Simile
Comparison using "like" or "as".
Example: "as brave as a lion"
Metaphor
Direct comparison without "like" or "as".
Example: "The moon was a ghostly galleon tossed upon cloudy seas."
Personification
Giving human qualities to non-human things.
Example: "The wind whispered through the trees."
Hyperbole
Extreme exaggeration for effect.
Example: "I'm so hungry I could eat a horse."

How to use this interactive view

  • Search — Use the search box above the table
  • Filter — Use dropdowns and text boxes in the filter row
  • Sort — Click any column header
  • Navigate — Use pagination buttons below the table
